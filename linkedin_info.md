# LinkedIn Project: Customer Churn Predictor

**Project name**
AI-Powered Customer Churn Analysis & Prediction Platform Using ML and Power Bi

**Description**
Developed an end-to-end predictive analytics solution to identify at-risk telecom customers using a high-performance ensemble engine (RandomForest, XGBoost, LightGBM, and CatBoost). The platform features a secure Flask backend and a premium, mobile-responsive web interface designed with a modern glassmorphism aesthetic for an intuitive user experience.

Additionally, I integrated Power BI to transform complex datasets into interactive dashboards, providing stakeholders with actionable business intelligence. By leveraging SMOTE to handle class imbalances and performing extensive feature engineering, the project demonstrates a complete integration of machine learning engineering and data visualization for effective churn management.

**Top 5 skills used in this project**
1. Machine Learning (Ensemble Modeling & SMOTE)
2. Data Visualization (Power BI)
3. Back-End Development (Python & Flask)
4. UI/UX Design (CSS Glassmorphism & Responsive Web Design)
5. Web Security (Secure Hashing, CSRF Protection, & Auth)
